# 水果挑選指南
## 組員 : 何睿純、李同岳

## 專案簡介
### 動機
台灣水果技術成熟、市場盛行且種類繁多，然而在挑選時經常覺得毫無頭緒。因此，我們以此做為發想，希望綜合長輩及專家的經驗，設計一個資訊平台，透過卡片式的版面與互動功能，展示常見水果的挑選技巧。  

### 核心功能
使用者可以瀏覽基本的選果原則、依類別篩選或搜尋特定水果，並點擊卡片查看詳細資訊，若有疑問可填寫聯絡表單。

## 使用技術
- **HTML5**：  
語意化標籤與內容分區。  
- **CSS3 / Bootstrap 5**：  
採用 Bootstrap 網格系統、預設元件與 RWD ，搭配少量自訂 CSS 打造一致的配色與 hover 效果。  
- **JavaScript**：  
使用原生 DOM API ( `querySelector` 、 `addEventListener` 、 `dataset` ) 操作頁面元素，實現 : 搜尋、篩選、折疊及表單驗證。  
- **Constraint Validation API**：  
表單驗證與自訂錯誤提示。  
- **Git 版控 / GitHub Pages** ：   
此專案已上傳至 GitHub，並透過 GitHub Pages 部署公開展示。

## 功能特色
1. **選果技巧整理**：  
整合專家與食品安全建議，例如 : 避免瘀傷、輕壓感受熟度、聞香辨甜度、選擇鮮豔當季水果等。

2. **卡片式水果介紹**：  
目前收錄蘋果、香蕉、莓果、柑橘、瓜類與梨子六種水果，每張卡片提供簡要挑選建議與延伸說明，內容引用自北達科他州立大學選果指南等。按下「更多資訊」即可展開細節。

3. **搜尋與篩選**：  
使用下拉選單與即時搜尋框，可依水果類別（仁果類、熱帶水果、莓果類、柑橘類、瓜類）或關鍵字快速篩選卡片。篩選邏輯透過 JavaScript 操作 `dataset` 實現。

4. **表單即時驗證**：  
聯絡表單包含姓名、電子郵件、主旨與訊息欄位，採用 HTML5 `required` 及 `type="email"` 限制，並透過 Constraint Validation API 及 Bootstrap 樣式顯示錯誤訊息，送出成功後給予提示並重置表單。

5. **響應式設計**：  
利用 Bootstrap 的 `col-lg-4 col-md-6` 配置讓卡片在大螢幕三欄、中螢幕兩欄、小螢幕單欄排列，各區段上下間距透過 `py-5`、`mb-3` 調整。

6. **額外互動效果**：  
自訂 `.hoverable` 與 `.btn-raise` 類別，為卡片與按鈕加入輕微位移與陰影，提升整體互動感。頁尾動態顯示當前年份。
